# 2.1. Maven no Visual Studio Code

## Introdução

Embora o Maven possa ser utilizado exclusivamente pela linha de comando sem nenhum problema, diversas IDEs oferecem algum tipo de integração com ele. O Visual Studio Code (VS Code) possui suporte ao Maven por meio de extensões que permitem executar comandos, gerenciar dependências e navegar pela estrutura dos projetos de forma visual, facilitando todo o processo de desenvolvimento de projetos Java.

Sendo assim, nessa aula extra vamos entender como configurar e usar o Maven no VSCode.

---

# Pré-requisitos

Para podermos configurar o Maven de maneira adequada no Visual Studio Code, é necessário já possuir tanto o JDK, quanto o Maven em seu computador. Para verificar se ambos estão instalados corretamente, rode os seguintes comandos:

``java -version``

``javac -version``

``mvn -v`` ou ``mvn -version``

Caso tudo esteja correto, uma mensagem especificando a versão do Java, do compilador Java e do Maven devem aparecer respectivamente. Caso contrário, recomendo que veja nossos tutoriais sobre a intalação do Java [aqui]() e sobre a instalação do Maven [aqui](02-Como_Instalar_o_Maven.md).

# Extensões Recomendadas

## Extension Pack for Java

O VSCode oferece diversas extensões para ajudar no desenvolvimento Java. Se você ja usa a IDE para desenvolvimento Java, deve possuir pelo menos uma delas instaladas.

Nós recomendamos a instalação do "Extension Pack for Java", como o  nome já sugere, ele é um pacote de extensões, ou seja, ao instalá-lo ele trará consigo várias extensões para o desenvolvimento Java, sendo elas:

- Language Support for Java™ by Red Hat;
- Debugger for Java;
- Test Runner for Java;
- Maven for Java;
- Gradle for Java;
- Project Manager for Java.

Se você já desenvolve a algum tempo, é possível que já conheça esse pacote de extensões ou mesmo que já o tenha instalado. Entre as extensões instaladas, a que mais importa para essa aula é a **"Maven for Java"**, que vai trazer funcionalidades do Maven para o VSCode.

## Maven for Java

Caso não queira instalar todo o pacote de extensões, você também pode instalar apenas a extensão **"Maven for Java"**. 

Entre as principais funcionalidades fornecidas pela extensão temos:

- Visualização da estrutura Maven;
- Navegação pelo `pom.xml`;
- Acesso rápido ao ciclo de vida do Maven;
- Gerenciamento de dependências.

---

## Abrindo um Projeto Maven

Com a extensão, basta abrir um projeto normalmente pelo VSCode. Caso o projeto aberto seja um projeto Maven, a extensão detectará o arquivo `pom.xml` automaticamente e o projeto passará a ser reconhecido como um projeto Maven.

## Criando um Projeto Maven

Com o Visual Studio Code aberto pressione as teclas `Ctrl + Shift + P` e procure por

```text
Maven: Create Maven Project
```
Com a opção selecionada, uma nova janela abrirá onde você pode selecionar entre os arquétipos do Maven (esse assunto ainda não foi abordado nas aulas, mas basta continuar acompanhando ele).Depois abrirá um novo espaço, onde você deve escrever o `groupId` do projeto e logo em seguindo o `artifact Id` (nome do projeto). Após isso, basta selecionar sua pasta e seu projeto estará criado!

[criar_projeto](resources/2.1-criar_projeto1.png)

Você também pode, com uma pasta aberta através do VSCode, clicar com botão direito, onde será possível ver uma nova opção chamada **Maven**, passe o mouse sobre ela e depois selecione a opção "New Project". 

[criar_projeto](resources/2.1-criar_projeto2.png)

Depois, basta preencher os campo e seu projeto estará criado!

## Explorador Maven

Com seu projeto Maven aberto, uma nova aba do **Maven** ficará disponível na barra lateral (logo abaixo de seus arquivos). A extensão reconhece projetos multi-módullo automaticamente, separando-os no explorador para facilitar a navegação. Nas seções do explorador você também tem controle sobre tudo do seus projetos, como: *lifecycle*, *plugins*, *dependencies*, *etc*.

[explorador_maven](resources/2.1-explorador_maven.png)
[explorador_maven2](resources/2.1-explorador_maven2.png)

## Executando Comandos Maven

Ao invés de rodar todos os comandos maven diretamente pelo terminal, dentro do seu explorador maven, na subseção *lifecycle*, você tem acesso aos principais comandos do Maven, bastando um clique para utiliza-los.

[comandos_maven](resources/2.1-comandos_maven.png)

Você também pode executar comandos Maven apertando `Ctrl + Shift + P`, pesquisando Maven e selecionando a opção `Maven: Execute Commands`. Depois basta selecionar qual comando você deseja rodar.


## Adicionando Depêndencias

Além de adicionar as depêndencias manualmente em seus `pom.xml` você pode, com o auxílio da extensão, adiciona-las em seu projeto pelo VSCode. Para isso, pressione as teclas `Ctrl + Shift + P` e procure pela opção `Maven: Add a dependency`, com ela selecionada procure pela dependência que deseja incluir, adicione ela e pronto! Simples, rápido e fácil!

Caso a depêndencia que procura não esteja aprecendo por algum motivo, você ainda pode adiciona-lá manualmente sem problemas.

Depois de adicionar as dependências, não esqueça de de rodar o comando ``mvn install`` para garantir que  elas foram instaladas corretamente!
